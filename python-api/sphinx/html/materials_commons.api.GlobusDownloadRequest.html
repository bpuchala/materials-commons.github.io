get_cli_transfer_source()[source]¶

A helper function to format the string that the Globus CLI expects for a transfer target; that is [endpoint-id]:[endpoint-path] .

Returns:String - the destination target suitable for the Globus CLI
get_url_transfer_source()[source]¶

A helper function to format the string that the Globus web app expects as a url encoding the destination target for transfer target; with the source unspecified.

Returns:String - URL for the Globus Web app; a transfer with the source unspecified, and the destination set to the target to Materials Commons
