delete()[source]¶

Delete this process. The process is removed from the database leaving only the local copy. A process can not be deleted if any of the following are true: (1) it is not a leaf node in the workflow, (2) is is in a dataset, (3) it is a create sample (type) process with one or more output samples.

Returns:id - the id of the process deleted or None (if the process was not deleted)

set_measurements_for_process_samples(measurement_property, measurements)[source]¶

Set a measurement for the indicated measurement_property.

Parameters:
  • measurement_property – dictionary - the property, see example below
  • measurements – mcapi.Measurement
Returns:

the updated Process value

 
>>> measurement_data = {
>>>     "name": "Composition",
>>>     "attribute": "composition",
>>>     "otype": "composition",
>>>     "unit": "at%",
>>>     "value": [
>>>         {"element": "Al", "value": 94},
>>>         {"element": "Ca", "value": 1},
>>>         {"element": "Zr", "value": 5}],
>>>     "is_best_measure": True
>>> }
>>> measurement = my_process.create_measurement(data=measurement_data)
>>>
>>> measurement_property = {
>>>     "name": "Composition",
>>>     "attribute": "composition"
>>> }
>>> my_process = my_process.set_measurements_for_process_samples(measurement_property, [measurement])
