class materials_commons.api.User.User(data=None)[source]¶

Note

normally created from the database by a call to top level function mcapi.get_all_users()

can_access(project)[source]¶

Does this user have permission to access the indicated project.

Parameters:project – an instance of the :class: mcapi.Project
Returns:boolean 
>>> from materials_commons.api import get_all_users
>>> user_list = get_all_users()
>>> for user in user_list:
>>>     if user.can_access(project):
>>>         print(user.fullname, user.email)
