Package mcapi - Top Level Functions¶

Project functions¶

mcapi.get_all_projects()[source]¶

Return a list of all the project to which the current user has access.

Example::

project_list = get_all_projects() for project in project_list:

print project.name

Creates a new Project object in the database and return it.

User function¶

mcapi.get_all_users()[source]¶

Return the list of all users registerd on the server.

Example::

user_list = get_all_users() for user in user_list:

print user.fullname, user.email
